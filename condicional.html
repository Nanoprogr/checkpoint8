  • ¿QUE ES UN CONDICIONAL?

    Piensa en los condicionales como las decisiones que tomas todos los días.

    Por ejemplo, si miras por la ventana y ves que está lloviendo, podrías decidir llevar un paraguas. En JavaScript, los condicionales son como si le dieras instrucciones a tu programa para que haga algo en función de ciertas circunstancias.


  • SINTAXIS

    la sintaxis básica de los condicionales en JavaScript es bastante sencilla.

    *if (condition) Aquí especificas la condición que quieres evaluar. Si esta condición es verdadera, se ejecuta el bloque de código dentro de las llaves {}


    EJEMPLO DE IF


    En este caso, el if determina si el numero es mayor que 5 y envia la impresion de es mayor que 5


    *else if (condition2) Si la primera condición no es verdadera, se evalúa esta segunda condición. Si es verdadera, se ejecuta el bloque de código correspondiente.


    EJEMPLO DE ELSE IF


    En este caso, el if determina si el numero es mayor que 15 y envia la impresion de es mayor que 15, else if dice que si el numero es mayor que 5 se envie la impresion de que es mayor que 5 pero menor que 15


    Como podemos intuir, en caso de que no se cumpla la primera condición (if) se establece una segunda condicion (el if) por lo que podemos decir efectivamente que si la primera condicion no se cumple pero la segunda si, pues el numero es menor a 15 ya que era la primera condicion que cumplir


    *else Este es el bloque que se ejecuta si ninguna de las condiciones anteriores es verdadera. No lleva una condición, ya que se ejecuta por defecto si ninguna de las anteriores se cumple.


    EJEMPLO DE ELSE


    En este caso, el if determina si el numero es mayor que 5 y envia la impresion de es mayor que 5, else nos dice que si no se cumple la condicion entonces se ejecute otro bloque de codigo, en este caso que envie la impresion de es menor o igual que 5



  • EN RESUMEN

    Un condicional en JavaScript es una estructura de control que permite tomar decisiones en función de ciertas condiciones. Esto significa que puedes ejecutar diferentes bloques de código dependiendo de si una condición específica es verdadera o falsa.
